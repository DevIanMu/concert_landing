# 盛夏焕新 · 港乐星夜 演唱会权益 H5 Demo

这是 Yunfeng Youyu「盛夏焕新 · 港乐星夜」演唱会权益 H5 落地页的可交互低保真原型。

## 在线访问

部署到 GitHub Pages 后，可通过以下地址访问：

```
https://DevIanMu.github.io/concert_landing/
```

## 文件说明

- `index.html`：H5 落地页可交互原型 v2.2（左上角场景切换器可切换 12 种演示场景：访客/未开户/老客/新客未达标/标准档/VIP 档/全部售罄/已过领票截止/已领票/已领票·待填收件信息/待填·已过截止/已结束；支持「稍后再填」先锁名额后补填；仅简体中文，数据为前端 mock，含 `activity_phase` 全局活动阶段）
- `design/state-machine.html`：状态机可视化
- `design/user-flow.html`：用户全链路流程图

## 本地预览

直接用浏览器打开 `index.html` 即可，无需构建步骤。

## 部署说明

本仓库已配置为 GitHub Pages 静态站点。推送后，在仓库 **Settings > Pages** 中选择分支即可启用。
